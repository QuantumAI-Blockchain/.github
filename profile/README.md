## Quantum Blockchain

**Physics-Secured Digital Assets with On-Chain AGI**

Quantum Blockchain is a post-quantum Layer 1 blockchain whose native currency is **Qubitcoin (QBC)**. It combines quantum computing (VQE mining), post-quantum cryptography (CRYSTALS-Dilithium, ML-KEM-768), supersymmetric economics (golden ratio emission), and an on-chain AGI reasoning engine (Aether Tree).
