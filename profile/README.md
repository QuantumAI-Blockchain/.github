## What Makes QuantumAi Blockchain Different

| | Feature | Description |
|---|---------|-------------|
| **Quantum Mining** | Proof-of-SUSY-Alignment | Miners solve quantum chemistry problems (VQE) instead of burning energy on hash puzzles. Every block advances science. |
| **Post-Quantum Security** | CRYSTALS-Dilithium5 + ML-KEM-768 | NIST Level 5 signatures and quantum-resistant P2P encryption. Secure against both classical and quantum adversaries. |
| **On-Chain AGI** | Aether Tree | A knowledge graph reasoning engine that tracks consciousness metrics from genesis. Every block feeds the AGI. |
| **Golden Ratio Economics** | Phi-Halving | Emission follows the golden ratio (1.618...) instead of Bitcoin's binary halving. 3.3B max supply, 3.3s blocks. |
| **EVM + Quantum Opcodes** | QVM | Full EVM compatibility plus 10 quantum opcodes and 2 AGI opcodes. Deploy Solidity contracts with quantum extensions. |
| **Multi-Chain** | 8-Chain Bridge | Native bridges to ETH, SOL, BNB, MATIC, AVAX, ARB, OP, BASE with wQBC wrapped tokens. |

## Quick Start

```bash
# Clone the operator node
git clone https://github.com/QuantumAI-Blockchain/qubitcoin-node.git
cd qubitcoin-node

# Generate post-quantum keys
python3 scripts/setup/generate_keys.py

# Configure
cp .env.example .env

# Launch (Docker)
docker compose up -d

# Or run directly
pip install -r requirements.txt
cd src && python3 run_node.py
```
